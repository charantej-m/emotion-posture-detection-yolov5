# Real-Time Emotion and Posture Detection using YOLOv5

## 📌 Project Overview

This project is a real-time emotion and posture detection system developed using YOLOv5 and OpenCV. It detects human emotions and body posture from live video streams, making it suitable for Smart Driver Assistance Systems (SDV), health monitoring, and human behavior analysis.

---

## 🚀 Features

- Real-time emotion detection
- Human posture detection
- YOLOv5 object detection
- Live webcam inference
- High-speed detection using OpenCV

---

## 🛠 Technologies Used

- OpenCV
- PyTorch
- NumPy
- Google Colab
