# Real-Time Emotion & Posture Detection Using YOLOv5  
A Computer Vision project for recognizing **emotions** and **postures** in real-time using custom YOLOv5 models.  
This system is designed for **in-vehicle passenger monitoring** to enhance safety in autonomous and smart vehicles.

---

##  Project Overview
This project uses a **custom-trained YOLOv5 model** to detect:
- Facial **emotions**
- Passenger **posture**
- Real-time movement patterns

The goal is to help identify unsafe passenger behavior and support intelligent vehicle safety systems.

---

##  Features
✔ Real-time Emotion Detection  
✔ Body Posture Recognition  
✔ Video and Webcam Inference  
✔ Custom YOLOv5 Training Pipeline  
✔ Dataset annotated using Roboflow  
✔ Model Deployment-ready (Jetson Nano support)
